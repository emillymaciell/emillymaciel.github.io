Pokédex Interativa com JavaScript e API

Pokedex interativa que consome a PokéAPI para carregar e exibir uma lista de Pokémon com paginação. A interface permite clicar em cada Pokémon para visualizar um painel de detalhes com suas estatísticas, habilidades e tipos. O design é responsivo, garantindo uma boa experiência de uso em diferentes tamanhos de tela.

  • Consumo da PokéAPI para carregar dados dos Pokémon de forma dinâmica;
  • Sistema de paginação para navegar pela lista completa;
  • Painel de detalhes com estatísticas, tipos e habilidades ao clicar em um Pokémon;
  • Design responsivo que se adapta a desktops e dispositivos móveis
